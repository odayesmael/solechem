# 🚀 Quick Start Guide

## ⚡ In 30 Seconds

```bash
# 1. Start the website
npm run dev

# 2. Open browser
# http://localhost:3000

# 3. Click "Products" in navigation
# See all 1,244 products!
```

---

## 🎯 What You Can Do Now

### 1️⃣ Browse Products
- Visit `/products` page
- See all 1,244 chemical products
- View complete product information

### 2️⃣ Filter by Category
- Click category dropdown
- Choose from 29 categories
- Products filter instantly

### 3️⃣ Filter by Industry
- Select from 16 industries
- Combine with category filters
- See relevant products only

### 4️⃣ Search Products
- Use search bar
- Search by name (e.g., "Methanol")
- Search by CAS number (e.g., "67-56-1")
- Search by formula (e.g., "CH4O")

### 5️⃣ View Product Details
- Click any product card
- See full specifications
- View safety information
- Check related products
